## Before You Start

<Steps>
  <Step title="Complete the Quickstart">
    Make sure ReasonDB is [installed and running](/quickstart) with your LLM configured.
  </Step>
  <Step title="Have an HTTP Client Ready">
    Tutorials use `curl` commands. Any HTTP client works.
  </Step>
  <Step title="Set Aside Focused Time">
    Each tutorial is designed to complete in one session.
  </Step>
</Steps>

## Tutorial Format

Every tutorial follows the same structure:

- **Goal**: What you'll build and why it matters
- **Sample Data**: Pre-made documents so you can follow along exactly
- **Step-by-Step**: Detailed walkthrough with explanations
- **Working Code**: Copy-paste commands that actually work
- **Challenges**: Optional exercises to deepen understanding

<Tip>
**Pro tip**: Run the commands as you read. Building muscle memory with real queries is more valuable than passive reading.
</Tip>
